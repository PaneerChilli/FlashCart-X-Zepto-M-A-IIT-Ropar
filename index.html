06 — INTEGRATION ROADMAP
Building the Moment Commerce Engine — Phase by Phase
AI TECHNOLOGY STACK — SPECIFIC MODELS
TFT
Temporal Fusion Transformer — Demand Forecasting
Per-dark-store model trained on SKU history + weather API + IPL/Holi/Ramzan calendar + salary-cycle patterns. 72-hr forecasts updated every 4hrs. ↓25% stockouts · ↓35% wastage · ₹18Cr/yr saved.
PPO/A3C
Deep RL VRP Solver — Rider Routing
Multi-order batching: 3 orders within 400m = 1 rider. Real-time HERE Maps. Learns Friday 6PM Linking Road jams. +18% orders/rider/hr · ₹3,760Cr/yr saved.
YOLO v8
Computer Vision — Dark Store Inventory
Edge cameras (NVIDIA Jetson Nano). Stockout detection in 3 sec. OCR reads expiry dates → auto-discount 48hr before. Fixes Zepto's FDA Dharavi violation issue. ↓60% wrong deliveries · ₹5Cr/day in refund prevention.
2-Tower
Two-Tower NN — Personalisation + Replenishment
User tower: brand loyalty, price sensitivity, household signals. Item tower: complementarity, margin profile. Predicts: "User needs atta in 4 days" → push notification. +8% order frequency · 34% replenishment conversion.
Bandit
Contextual Bandit — Ethical Occasion Pricing
Zone-level demand surge pricing only (NOT device-type — kills Zepto's dark pattern). 3× premium pricing for Holi/IPL/Diwali occasion slots for brands. Transparent to consumers. Trust rebuild → ↓5% churn = ₹200Cr/yr retained GMV.
Llama 3.1
Fine-tuned Indic LLM — Tier 2/3 WhatsApp Commerce
Fine-tuned on 22 Indian languages. WhatsApp: "Ek kilo pyaaz aur doodh bhejo" → parsed → SKU matched → UPI link. 530M WhatsApp users. Zero app install friction. Unlocks $800M–1.2B incremental GMV from 50+ new cities.
